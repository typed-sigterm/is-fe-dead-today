Frontend Engineering is dead today because:

1. **"I design with Claude more than Figma now"**: Development workflows have shifted decisively. Engineers are no longer manually building UI components or laboring over Figma mockups. Instead, they are using generative models like Claude Code to go directly from ideation to functional, production-ready interfaces. The traditional role of the "Frontend Developer" as an implementer of designs is being subsumed by AI-driven, intent-based creation.
2. **Emergence of Agentic Development**: With the release of sophisticated tools like Hermes Agent v0.16.0 and the broader trend toward "Agentic PCs," we are moving away from manual coding entirely. These agents are now capable of end-to-end software engineering workflows—including UI construction and testing—making the manual assembly of the DOM a legacy practice.
